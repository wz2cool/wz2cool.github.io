前言

上一章我们我们建立好 SonarQube 入门（一）：Docker 版本安装, 这里我们简单介绍一下如何创建一个项目吧，这个每次我找都要找半天，方便自己记录一下啊。

步骤

  1. 到右上角点击我的账号
  2. 点击安全选项卡
  3. 填项目名字，并生成 token， 这里我填写的是 test-project
  4. 复制 token 去跑一下 maven 项目,login 参数就是刚才的 token
